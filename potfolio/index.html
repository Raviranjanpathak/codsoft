Simon Says – Memory Game
I built a fully functional Simon Says memory game using HTML, CSS, and vanilla JavaScript. The game challenges users to memorize and repeat an increasingly complex sequence of colors and sounds. It features responsive design, smooth animations, and interactive feedback through visual cues and audio. The game logic handles sequence generation, user input validation, level progression, and restart functionality. This project helped strengthen my understanding of JavaScript event handling.

Image Resizer

Image Resizer Web App
I developed a responsive Image Resizer web application using HTML, CSS, and vanilla JavaScript. This tool allows users to upload an image, input custom dimensions, and instantly resize it directly in the browser without any need for server-side processing. Users can preview their changes in real time and download the resized image with a single click. The project includes input validation to ensure only supported image formats are used, and the interface is optimized for both desktop and mobile devices.
